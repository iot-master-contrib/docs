# 智能网关

智能网关是物联大师的主要应用场景，适用于物联网现场使用，相较于服务器有以下特点：

  1. 实时高效
  2. 避免网络异常导致失控
  3. 发挥边缘计算能力，减轻服务器压力

# 物联大师管理中心

物联大师管理中心是物联大师的统一管理平台，客户可以云端部署，统一管理所拥有的智能网关，主要功能有：

  1. 网络桥接，外网访问物联大师
